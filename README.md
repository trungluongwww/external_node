## Project repository

This repository provides a set of tools and configurations to help you set up your Node.js project quickly.

### Getting started

1. Open terminal and navigate to the root directory of your project
2. Add this repository as a submodule by running this cmd
   ``git submodule add <URL_REPOSITORY>``

### Installation

1. Once you have added the repository as a submodule. navigate to submodule directory
   ``$cd external_node``
2. install the required dependencies by running the following command:
   ``$npm install``
